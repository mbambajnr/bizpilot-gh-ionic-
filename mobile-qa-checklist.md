## Severity Guide

- `Blocker` - cannot complete core task
- `Major` - task completes but UX is broken, confusing, or risky
- `Minor` - noticeable issue but still usable

---

## Evidence Rules

If something fails, capture:

- screenshot
- short screen recording for keyboard / scroll / touch / share / download issues
- page / flow
- exact step
- severity
- short notes
